DriverAtributInt
En aquest driver es proven totes les funcionalitats de la classe AtributInt.

Hi ha inclos un joc de proves anomenat: input_jproves.txt

Prova totes les funcionalitats de la classe que funcionin correctament i com es esperat detectant els possibles errors:
 - Crea un atribut amb nom atributint de tipus boolean i despres de tipus numeric per a que retorni error de tipus
 - Crea un atribut amb nom atributint de tipus float.
 - Consulta el nom amb el getter
 - Consulta el tipus amb el getter
 - Canvia el nom del atribut per horesEsport amb el setter
 - Canvia el tipus de l'atribut amb el setter
 - Comprova si un valor esta dins un interval ( aprovat [5..10]).
 - Consultem el nom del interval al que pertany un valor provant si no pertany abans.
 - S'afegeix un interval al atribut.
 - S'afegeix una llista d'intervals.
 - Consulta la llista d'intervals de l'atribut.

Per executar el driver, utilitzar la comanda: java -jar Driver_AtributInt.jar

Per utilitzar el joc de proves: java -jar Driver_AtributInt.jar < input_jproves.txt
