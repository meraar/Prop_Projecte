Driver ReglaAssociacio
En aquest driver es proven totes les funcionalitats de la classe ReglaAssociacio.

Hi han un joc de prova:

input_jprova.txt:
Prova totes les funcionalitats de la classe que funcionin correctament i com es esperat:
 - Crea una regla d'associacio amb aquests valors:
        -Identificador: 1
        -Confianca: 0.3
        -Suport: 0.4
        -Condicions: alcohol = 0.5, distret = cert
        -Consequent: risc = alt
 - Canvia l'identificador amb el setter
 - Consulta l'identificador amb el getter
 - Canvia la confianca amb el setter
 - Consulta la confianca amb el getter
 - Canvia el suport amb el setter
 - Consulta el suport amb el getter
 - Canvia el llistat de condicions el setter
 - Consulta el llistat de condicions amb el getter
 - Canvia el consquent amb el setter
 - Consulta el consequent amb el getter
 - Comprova que es compleixi la regla d'associacio

Per executar el driver, utilitzar la comanda: java -jar Driver_ReglaAssociacio.jar

Per utilitzar el joc de proves 1: java -jar Driver_ReglaAssociacio.jar < input_jprova.txt
